        (if (< x y) x y))
    (define (min_three x y z)
        (min_two x (min_two y x)))
    (define (sum_of_larger_two x y z)
        (cond ((= x (min_three x y z)) (+ y z))
              ((= y (min_three x y z)) (+ x z))
              ((= z (min_three x y z)) (+ x y))))

> Ex1.4

`(if (> b 0) + -)`该组合式的返回值是一个函数，如果组合式`(> b 0)`返回值为`#t`，则返回函数`+`，否则返回函数`-`。

> Ex1.5

**Normal Order（正则序）**: 先展开再归约，即惰性求值。

**Applicative Order（应用序）**: 先求值参数然后应用。

本例中如果采用应用序，那么会先求出组合式`(p)`的值然后陷入无限递归。如果采用正则序，会将表达式展开为`(if (= x 0) 0 (p))`，由于`if`的谓词部分返回为`#t`，所以会直接返回0，而不会对*<alternative>*部分进行求值。
