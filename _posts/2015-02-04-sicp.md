##Chapter 1.1

**bound variable**: The procedure definition *binds* its formal parameters. The meaning of a procedure definition is unchanged if a bound variable is consistently renamed throughout the definition. If a variable is not bound, we say that it is *free*. The set of expressions for which a binding defines a name is called the *scope* of that name.

**lexical scoping**: The `x` gets its value from the argument with which the enclosing procedure `sqrt` is called. 

<pre class="brush:erl">
(define (sqrt x)
  (define (good-enough? guess)
    (< (abs (- (square guess) x)) 0.001))
  (define (improve guess)
    (average guess (/ x guess)))
(define (sqrt-iter guess)
    (if (good-enough? guess)
        guess
        (sqrt-iter (improve guess))))
  (sqrt-iter 1.0))

> Ex1.5

**Normal Order（正则序）**: 先展开再归约，即惰性求值。

**Applicative Order（应用序）**: 先求值参数然后应用。

本例中如果采用应用序，那么会先求出组合式`(p)`的值然后陷入无限递归。如果采用正则序，会将表达式展开为`(if (= x 0) 0 (p))`，由于`if`的谓词部分返回为`#t`，所以会直接返回0，而不会对*<alternative>*部分进行求值。

> Ex1.6

<pre class="brush:erl; highlight: 4">
    (define (sqrt-iter guess x)
        (if (good-enough? guess x)
        guess
        (sqrt-iter (improve guess x)
                    x)))
</pre>

根据`P12`上关于if表达式的定义，只有当谓词为假时才回去求值`<alternative>`表达式，否则不会。而本例中使用if语句的前提是当谓词为真时结束递归，不再调用第4行的`sqrt-iter`递归。而`new-if`为普通函数，在解释器使用*应用序*求值时，`<then-clause>`和`else-clause`都会被求值，这会造成无限递归。

##Chapter 1.2

**Recursive process**: The expansion occurs as the process builds up a chain of *deferred operations* (in this case, a chain of multiplications). The contraction occurs as the operations are actually performed.

**Linear recursive process**: the length of the chain of deferred multiplications, and hence the amount of information needed to keep track of it, grows *linearly* with `n` (is proportional to `n`), just like the number of steps. 

![](http://mitpress.mit.edu/sicp/full-text/book/ch1-Z-G-7.gif)

**Iterative process**: an iterative process is one whose state can be summarized by a fixed number of *state variables*, together with a fixed rule that describes how the state variables should be updated as the process moves from state to state and an (optional) end test that specifies conditions under which the process should terminate.

**Linear iterative process**: In computing `n!`, the number of steps required grows *linearly* with n. Such a process is called a linear iterative process.

![](http://mitpress.mit.edu/sicp/full-text/book/ch1-Z-G-10.gif)

**Tail-recursive**: It will execute an iterative process in constant space, even if the iterative process is described by a recursive procedure.

> Ex 1.10

Ackermann函数是德国数学家Wilhelm Ackermann发现的一个递归却非原始递归的函数。详细的定义可以参考[这里](http://en.wikipedia.org/wiki/Ackermann_function)。这里根据该函数的lisp定义可以写出该函数的数学表达式。

$$A(x,y) = \begin{cases}
    0 & y = 0\\
    2y & x = 0\\
    2 & y = 1\\
    A(x - 1, A(x, (y - 1))) & otherwise
\end{cases}$$

由上式不难推出

(A 1 10) = $$2^{10}$$  
(A 2 4) = $$2^{16}$$  
(A 3 3) = $$2^{16}$$  

以下所有函数定义域均为自然数集。

$$f(n) = 2n$$  
$$g(n) = \begin{cases} 
    0 & n = 0\\
    2^n & otherwise
\end{cases}$$  
$$h(n) = \begin{cases}
0 & n = 0\\
2 & n = 1\\
2^{h(n - 1)} & otherwise
\end{cases}$$

> Ex 1.13

复习高中数学……$$a_1 = 1, a_2 = 1, a_n = a_{n-1}+a_{n-2} (n \geq 3)$$

构造等比数列：
$$a_{n+1}+xa_{n} = y(a_n+xa_{n-1})$$

带入$$a_{n+1} = a_n + a_{n-1}$$

得到一组解
$$\begin{cases} 
x = \frac{\sqrt{5}-1}{2}\\
y = \frac{\sqrt{5}+1}{2}
\end{cases}$$

设$$b_n = a_{n+1}+\frac{\sqrt{5}-1}{2}a_{n}$$

可以得到$$b_n = a_{n+1}+\frac{\sqrt{5}-1}{2}a_{n} = (\frac{\sqrt{5}+1}{2})^n$$

再次构造等比数列$$a_{n+1}+x(\frac{\sqrt{5}+1}{2})^{n+1} = (\frac{1-\sqrt{5}}{2})(a_n+x(\frac{\sqrt{5}+1}{2})^n)$$

得到$$x = -\frac{\sqrt{5}}{5}$$

设$$c_n = a_n - \frac{\sqrt{5}}{5}(\frac{\sqrt{5}+1}{2})^n$$

则$$c_n = a_n - \frac{\sqrt{5}}{5}(\frac{\sqrt{5}+1}{2})^n = -\frac{\sqrt{5}}{5}(\frac{1-\sqrt{5}}{2})^n$$

可以得到$$a_n = \frac{\sqrt{5}}{5}[(\frac{\sqrt{5}+1}{2})^n - (\frac{1-\sqrt{5}}{2})^n]$$

$$\|a_n - \frac{\phi^n}{\sqrt{5}}\| = \|\frac{\sqrt{5}}{5}(\frac{\sqrt{5}-1}{2})^n\| < \|\frac{5-\sqrt{5}}{10}\| < \frac{1}{2}$$，所以是最接近$$\frac{\phi^n}{2}$$的整数。
